# /game

Create a browser-based game.

## Usage
```
/game <description>
```

## Examples
```
/game platformer where you collect coins
/game space shooter with asteroids
/game quiz game about world capitals
/game roguelike dungeon crawler
/game 3d racing game
/game memory matching card game
/game breakout clone
/game flappy bird clone
/game tower defense
/game snake game
```

## What Happens
1. Analyzes your game concept
2. Selects appropriate framework (KAPLAY.js for 2D, Three.js for 3D)
3. Implements the game with proper game feel (juice, feedback, polish)
4. Playtests in headless browser
5. Critiques and fixes any issues
6. Launches the result

## Arguments
- `description`: What kind of game to create

The agent will create a playable game with:
- Menu screen
- Core gameplay
- Win/lose conditions
- Score tracking
- Restart functionality
- Control hints
