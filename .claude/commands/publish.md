# /publish
This command triggers the **publisher** skill to publish visualizations created by the **salvador** skill to configured remote destinations.

## Usage
`/publish [visulaization_name]`

## Behavior
1.  Immediately invoke the `publisher` skill.
2.  Pass the `[visulaization_name]` argument as the visualization to publish.
