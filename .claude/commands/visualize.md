# /visualize
This command triggers the **salvador** skill to visualize the provided concept.

## Usage
`/visualize [concept]`

## Behavior
1.  Immediately invoke the `salvador` skill.
2.  Pass the `[concept]` argument as the topic to visualize.
3.  Ensure the autonomous "Implement -> Inspect -> Refine" loop is used.
